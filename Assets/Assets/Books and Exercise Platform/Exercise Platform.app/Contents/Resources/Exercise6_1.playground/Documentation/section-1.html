Assume that the first element of the array is also the largest.

var listOfNumbers = [1, 2, 3, 10, 100]

var maxVal = listOfNumbers[0]

for number in listOfNumbers {
    if maxVal < number {
        maxVal = number
    }
}

print(maxVal)

To get the maximum number from a list of numbers we are going to take each number and remember the biggest so far in temporary variable. After we look ar each of the numbers that variable will have the bigest one of them.
