Count the number of divisors of the input number.

var number = 17

var numberOfDivisors = 0

for i in 1...number {
    if number % i == 0 {
        numberOfDivisors += 1
    }
} 

if numberOfDivisors == 2 {
    print("prime")
} else {
    print("not prime")
}

We count the number of divisor of number and store the result in numberOfDivisors. If numberOfDivisors if 2 then the number is prime.
