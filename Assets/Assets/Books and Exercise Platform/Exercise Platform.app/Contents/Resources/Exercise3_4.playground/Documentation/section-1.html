Converting a Double to an Int discards all the digits after the decimal point. 

var number = 5.1517

var intNumber = Int(number * 10.0)

var roundedNumber = Double(intNumber) / 10.0

Converting a Double to an Int will discard all the digits after the decimal point we want to keep a digit tough so we'll multiply the number by 10 first then we'll discard all the digits by converting to Int, finally we'll convert back to Double and divide by 10 to shift the decimal point to the right.

Input Expected values 
var number = 5.1517

roundedNumber = 5.1

var number = 32.5

roundedNumber = 32.5

var number = 2.0

roundedNumber = 2.0
