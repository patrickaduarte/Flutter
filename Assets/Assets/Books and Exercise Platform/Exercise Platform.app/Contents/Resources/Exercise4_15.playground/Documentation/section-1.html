The smallest divisor of a and b is 1. And the greatest value can be at most min(a, b).

Find the minimum of a and b and store it in maxDiv.
Write a for loop that goes from 1 to maxDiv and check each number.

var a = 24
var b = 18

var maxDiv = a

if b < maxDiv {
    maxDiv = b
}

var gcd = 1

for i in 1...maxDiv {
    if (a % i == 0) && (b % i == 0){
        gcd = i
    }
}

print(gcd) // 6

The smallest divisor of a and b is 1. And the greatest value can be at most min(a, b), this will be stored in maxDiv. Using a loop we check each number from 1 to maxDiv as a posible common divisor of a and b.
