The order in which you apply the map, filter and reduce operations is important.

var numbers = [1, 2, 3, 4, 5, 6]

let sum = numbers.filter {
        $0 % 2 == 1 //select all the odd numbers
    }.map {
        $0 * $0 // square them
    }.reduce(0, +) // get their sum

print(sum)

To solve this problem we shoudl solve each step separately. First of all let's get all the odd numbers from numbers: this is similar to the multiples of 3 problem. We are goin to use filter and a closure that returns true if a number is odd. Then we have to square all the numbers. We can do this using map. In the end we need to add all the numbers - this can be done with .reduce(0, combine: +).
