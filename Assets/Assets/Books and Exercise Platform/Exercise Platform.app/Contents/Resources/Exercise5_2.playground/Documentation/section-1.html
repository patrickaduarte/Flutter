Use string interpolation.

var a = 14
var b = 23

var sum = a + b

var formattedSum = "\(a) + \(b) = \(sum)"

This problem is best solved via string interpolation. We just need to format our interpolated string correctly. The sum can be stored in a variable or computed directly in the interpolared string ("\(a) + \(b) = \(a + b)").

Input Expected values 
var a = 14
var b = 19

formattedSum = "14 + 19 = 31"

var a = 12345
var b = 98765
