To get the digits of a number you need to get the digits of the number without its last digit (divide by 10). And then add the last digit to that result.

func digits(_ number: Int) -> [Int] {
    if number >= 10 {
        let firstDigits = digits(number / 10)
        let lastDigit = number % 10
        return firstDigits + [lastDigit]
    } else {
        return [number]
    }
}

We can get the last digit of a number by computing it's remainder to 10. We can remove the last digit by dividing the number by 10. If we take the last digit, remove it and repeat until the number becomes 0 we get the digits in reverse order. We can use recursion to reverse it - first we get the digits of the number without the last then we add the last digit.
